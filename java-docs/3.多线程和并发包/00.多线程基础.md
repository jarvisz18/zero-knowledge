sleep和wait的区别
1.sleep是线程类的方法，wait是Object的方法
2.sleep不会释放锁，wait方法会释放锁
3.sleep方法不依赖与monitor锁，而wait方法需要
4.sleep方法不需要唤醒，而wait方法需要
