A. epoll 和 select 都是 I/O 多路复用的技术，都可以实现同时监听 多个I/O事件的状态。

B. epoll 相比 select 效率更高，主要是基于其操作系统支持的 I/O 事件通知机制，而select是基于轮询机制。

C. epoll支持水平触发和边沿触发两种模式。

D. select能并行支持I/O比较小，且无法修改。
